## Overview
RiddleMaster is an **interactive web-based riddle game** built with **Spring Boot**, **Thymeleaf**, **MySQL**, and **Docker**. Users can solve riddles, submit answers, and track their scores on a leaderboard. The project demonstrates full-stack Java web development, REST API design, database integration, and containerized deployment.

## Features
- Web interface built with **Thymeleaf** for interactive riddle solving.
- **REST APIs** for managing riddles, users, and leaderboard data.
- Persistent storage using **MySQL** with **JPA/Hibernate**.
- **Dockerized application** for easy deployment on any environment.
- Leaderboard tracking to display top-performing users.
- Modular and scalable architecture suitable for enhancements.

## Tech Stack
- **Backend:** Java, Spring Boot, RESTful APIs
- **Frontend:** Thymeleaf templates
- **Database:** MySQL, JPA/Hibernate
- **Containerization:** Docker
- **Build & Dependency Management:** Maven
